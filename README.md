# auditah

Obligation-aware license compliance + attribution tool for gamedev.

A license is not an identifier — **a license is a set of obligations and permissions.** auditah stores attribution + license info in sidecars that travel with each asset, audits whether each obligation is fulfilled, and generates `CREDITS.md` from the result.

See `.plans/auditah/plan.md` for the full specification.
